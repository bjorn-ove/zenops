# zenops-safe-relative-path-validator

Internal helper crate for [zenops](https://crates.io/crates/zenops). Holds the
shared traversal-validation function used by both
[`zenops-safe-relative-path`](https://crates.io/crates/zenops-safe-relative-path)
and
[`zenops-safe-relative-path-macros`](https://crates.io/crates/zenops-safe-relative-path-macros).
