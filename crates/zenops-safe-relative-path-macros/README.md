# zenops-safe-relative-path-macros

Internal helper crate for [zenops](https://crates.io/crates/zenops). Provides
the `srpath!()` proc-macro used by
[`zenops-safe-relative-path`](https://crates.io/crates/zenops-safe-relative-path)
to validate relative-path literals at compile time.
